# Game Assets

## Sprites Directory

Place character sprite sheets here.

### Malachar Sprite Sheet
- File: `sprites/malachar.png`
- Format: 8 columns x 8 rows sprite sheet
- Frame size: ~64x64 pixels per frame
- Walking animations in rows

### Other Character Sprites
You can add sprite sheets for other characters:
- `warrior.png`
- `mage.png`
- `rogue.png`
- `archer.png`
- `paladin.png`
- `necromancer.png`

## Format
All sprite sheets should follow the same format for consistency.
